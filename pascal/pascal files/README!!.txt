EXPLICAÇÃO DO EXERCÍCIOS

contale2: Conta lentras distintas de um cadeia, como por exemplo, na 	  palavra "BANANA" temos 3 letras distintas(a,b,n).

contarle: Conta uma letra escolhida pelo usuário dentro de uma cadeia 	  qualquer.

contarpa: Conta o nº de palavras de uma frase qualquer.

exe31: Diz respeito a exercício 31 da apostila 2. Analogamente exe33,34        e 35.

exematriz: Exemplo de alimentação e exibição de uma matriz qualquer.

função1: Calcula a média e o desvio padrão de um conjunto números,              usando função.

inserord: Coloca inteiros em ordem crescente.

interv1: Inverte uma cadeia.

invert2: Inverte uma cadeia criando outra cadeia.

maiomenf: Mostra o maior e o menor nº de um vetor usando função.

maiormat: Mostra o maior valor de uma linha de uma matriz.

matrizin: Diz se a matriz é ou não uma matriz identidade.

matsimet: Informa se a matriz é simétrica ou não.

mesclar: Mescla dua cadeias a partir de uma ordem definida pelo 	 usuário.

ordena:	Ordena nomes em ordem alfabética, para tanto pode usar dois	        métodos: o da bolha ou seleção.

polinomi: Calcula e exibe o valor de um polinômio de grau definido pelo 	  usuário.

proced1 e 2: Usa procedimento para ordenar nomes e transformar 	     	     minúculos em maísculos respectivamente.

prodmatr: Produto de matrizes.

registro: Calcula mediana de salários usando registro.

somacolu: Soma colunas de uma matriz.

somamatr: Soma matrizes.

trocalet: Troca uma letra de uma cadeia a partir de uma sílaba definida 	  pelo usuário.

trocanx: Troca a letra de posição "n" pela letra  de posição "x".
